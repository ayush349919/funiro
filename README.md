Furniro - Modern Furniture E-Commerce Platform
Furniro is a high-end, responsive e-commerce web application designed for a premium furniture shopping experience. It features dynamic product listing, advanced filtering, and a robust cart management system powered by Redux.
🚀 Key Features
Premium Landing Page: A responsive Hero section with high-impact visuals and clear CTAs.
Dynamic Product Catalog: Real-time data fetching with features to filter by category and sort by price (High/Low).
Centralized Cart System: A fully functional shopping cart using Redux Toolkit to manage quantities, totals, and persistence.
Interactive UI/UX: Built with Tailwind CSS for mobile-first responsiveness, featuring smooth hover effects and loaders.
Brand Story & Social Proof: Dedicated sections for company milestones and "About Us" information.
Contact Management: A validated contact form with real-time state handling for customer support.
🛠️ Tech Stack
Frontend: React.js
State Management: Redux Toolkit (Thunks for Async Logic)
Styling: Tailwind CSS
Navigation: React Router DOM
Icons: Lucide-React
Fonts: Google Fonts (Poppins)
